async await

发表于 2019-08-21 | 分类于 javascript | 阅读次数:

基本用法

async 函数返回一个 Promise 对象，可以使用 then 方法添加回调函数。
当函数执行的时候，一旦遇到 await 就会先返回 -> 等到异步操作完成 -> 再接着执行函数体内后面的语句。

多种使用方式

错误处理机制

await 命令后面的 Promise 对象如果变为 reject 状态，则 reject 的参数会被 catch 方法的回调函数接收到。

  • !! 任何一个 await 语句后面的 Promise 对象变为 reject 状态，那么整个 async 函数都会中断执行。如果想继续执行介意再用 try catch 包起来

执行

async 返回一个 promise 对象，必须等内部所有 await 命令执行完毕 或 遇到 return 或 抛出错误，才会发生状态改变。、async 函数内部的异步操作执行完，才会执行 then 方法指定的回调函数。

同时触发

多个 await 命令后面的异步操作，如果不存在继发关系，最好让它们同时触发.
