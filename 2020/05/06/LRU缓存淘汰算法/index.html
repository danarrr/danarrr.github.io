什么是 LRU 算法

其实就是一种缓存淘汰策略，在内存有限的情况下，淘汰最近且最少使用的数据或者页面。

缓存在计算机网络经常看到，比如首次打开一个网页，加载会非常缓慢，第二次打开就会加速了。

这就是 LRU 算法在浏览器上的应用，
说几种比较常见的淘汰策略，比如 FIFO(先进先出)、LFU(最少使用)、LRU(最近最少使用)。

主要实现这三个内容：

  1. 如果缓存内存this.cache不存在用户现正访问的 tab，则将新浏览的 tab Node 节点保存到 key 值中，并添加到this.cache最后一位.
  2. 如果缓存内存this.cache存在用户现正访问的 tab，则找出这个节点的 key 删除，并添加新的内容。
  3. 如果超过缓存内存的最大值this.max, 则从this.cache开始淘汰最久最少使用的数据
